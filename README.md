# go-ekit
## go的泛型工具类

#### 一个G0泛型工具库，用于重构在业务代码中的冗余、模板代码。该工具库提供了高性能的辅助方法和数据结构，包括:
    ·切片的辅助方法:添加、删除、查找，求并集、交集
    ·map辅助方法
    ·扩展map实现:接受任意类型的HashMap
    ·协程池

### 更多内容逐步补充中
    ·扩展map实现: TreeMap, LinkedMap
    ·List 实现: LinkedList、 ArrayList 和 SkipList
    ·Set: 包括 HashSet 和 TreeSet, SortedSet
    ·队列:普通队列、优先级队列
    ·bean操作辅助类:高性能高扩展的bean copier机制
    ·并发扩展工具:包括并发队列、并发阻塞队列、并发阻塞优先级队列
